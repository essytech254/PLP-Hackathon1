Education Inequality Analysis

This project focuses on analyzing global educational inequality trends from 2010 to 2021. It features a Power BI dashboard that visualizes the disparities in education access and quality across different regions and demographics.

The dashboard includes interactive charts and graphs, allowing users to explore the data dynamically. Using a dataset from Kaggle, the dashboard provides insights into how inequality varies by region, year, and human development group.

It aims to support SDG 4 initiatives by helping researchers and investors identify disparities and target improvement areas.
